# Firstbrain Vault

## Goal

> Ein AI-natives Wissensmanagement-System aufbauen, das als Second Brain fungiert. Claude Code als kognitive Schicht, Obsidian als Speicher. Das System soll sich selbst verbessern und wachsen.

## Context

- **Why:** Wissen geht verloren wenn es nicht systematisch erfasst wird. KI kann die organisatorische Arbeit übernehmen.
- **Deadline:** Kein festes Ende -- ongoing, aber v1.2 shipped
- **Stakeholder:** Ich selbst, Open Source Community (GitHub: BEKO2210/Firstbrain)

## Notes

Das Projekt ist gleichzeitig das Tool und der Gegenstand. Meta, aber produktiv. Jede Verbesserung am Vault-System verbessert auch die eigene Produktivität.

12 Skills, 12 Templates, 9 MOCs, 4-Layer Memory System. Alles zero-dependency (nur Transformers.js optional für Semantic Search).
