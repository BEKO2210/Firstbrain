# Softwareentwicklung

## Description

> Kernbereich meiner beruflichen Tätigkeit. Umfasst alle Softwareprojekte, technischen Entscheidungen, Tool-Evaluierungen und Architektur-Patterns.

## Standards & Goals

- Clean Code und wartbare Architekturen bevorzugen
- Lokale KI-Lösungen evaluieren und einsetzen (Privacy-First)
- Open Source bevorzugen wo möglich
- Technische Schulden regelmäßig abbauen
- Neue Tools und Frameworks systematisch bewerten

## Habits & Routines

- Wöchentliches Code-Review der eigenen Projekte
- Monatliche Tool-Evaluierung: Was gibt es Neues?
- Zettel schreiben für jedes neue Konzept das ich lerne
