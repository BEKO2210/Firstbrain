# Atomare Notizen schreiben

## Idee

> Eine atomare Notiz enthält genau einen Gedanken, vollständig und in eigenen Worten formuliert — so, dass sie ohne Kontext verständlich ist.

## Erklärung

Das Konzept der atomaren Notiz ist das Fundament des Zettelkasten-Systems. "Atomar" bedeutet: unteilbar. Eine Notiz sollte sich auf einen einzigen, klar abgrenzbaren Gedanken beschränken. Nicht ein ganzes Kapitel zusammenfassen, nicht drei Ideen in einen Text packen, sondern eine Idee isolieren und sie so formulieren, dass sie für sich allein steht.

Der Grund dafür ist Verknüpfbarkeit. Eine Notiz mit fünf verschiedenen Ideen lässt sich schlecht verlinken — worauf genau verweist der Link? Eine atomare Notiz dagegen ist ein präziser Baustein, der sich mit anderen Bausteinen kombinieren lässt. Je kleiner und klarer die Einheit, desto flexibler ist sie einsetzbar.

Wichtig dabei: Atomar heißt nicht kurz. Eine atomare Notiz kann durchaus mehrere Absätze umfassen, solange sie einen einzigen Gedanken entwickelt. Die Länge richtet sich nach der Idee, nicht nach einer Zeichenvorgabe.

## Beispiel

**Atomar:** "Progressive Summarization funktioniert in mehreren Durchgängen: Beim ersten Lesen markiere ich grob, bei jedem weiteren Besuch verdichte ich weiter."

**Nicht atomar:** "Tiago Forte erklärt PARA, CODE und Progressive Summarization. PARA steht für Projects, Areas, Resources, Archive. CODE bedeutet Capture, Organize, Distill, Express. Progressive Summarization hebt stufenweise hervor." — Das sind drei verschiedene Konzepte in einer Notiz.

**Nicht atomar:** Ein Screenshot eines Artikels ohne eigene Formulierung — das ist eine Sammlung, keine Notiz.
