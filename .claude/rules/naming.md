File naming conventions for all vault folders:

- Projects: Descriptive name -> "Website Relaunch.md"
- Areas: Area name -> "Health.md", "Finances.md"
- Resources: Prefix + name -> "Book -- Title.md", "Tool -- Name.md", "Zettel -- Idea.md"
- Daily Notes: Date -> "YYYY-MM-DD.md"
- Decisions: "Decision -- Topic.md"
- Meetings: "Meeting -- Topic YYYY-MM-DD.md"
- People: "Person -- Name.md"
- Code Snippets: "Snippet -- Name.md"
- All filenames in English
- Use " -- " (space-dash-dash-space) as separator, not em-dash
